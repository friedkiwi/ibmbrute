# ibmbrute 

A password brute-force tool for SST/DST users on IBM AS/400 systems.

This tool supports both multi-threaded CPU based cracking, Apple Metal-based cracking and NVIDIA CUDA-based cracking. 

## Usage:

```bash
./ibmbrute --help
```/fa
